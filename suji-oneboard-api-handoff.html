👤 담당: 권수지 대리

  4. 키는 절대 카톡/공개 채팅에 올리지 말 것 (보안)
  5. P1 완료되면 P2 (카페24), 그 다음 P3·P4 순서

🟢 P1 · 네이버 스마트스토어

먼저 시작
P1

네이버 커머스 API 센터 앱 등록

소요시간 10분 · 난이도 ⭐
필요한 키 2개 — client_id · client_secret
1
네이버 커머스 API 센터 접속 → apicenter.commerce.naver.com
🔗 API 센터 열기
2
스마트스토어 운영 계정으로 로그인
3
상단 "애플리케이션 등록" 클릭 → 새 앱 만들기
4
앱 이름: OneBoard 연동 입력
앱 용도: 매출/주문 조회용
Redirect URL: 나중에 찰스가 알려줌 (일단 https://oneboard-server.onrender.com/auth/callback 입력)
5
권한 스코프 체크: 판매자 주문 조회 ✓, 상품 조회 ✓, 매출 통계 ✓
6
등록 완료 → "내 애플리케이션" 페이지에서 client_id 와 client_secret 복사
7
찰스에게 전달 (Gmail 또는 잔디 DM — 카톡/공개 채팅 금지)
⚠️ 보안 주의: client_secret 은 비밀번호처럼 다뤄야 합니다. 화면 캡처, 카톡, 공개 문서에 절대 올리지 마세요. 유출 시 재발급 가능합니다.

🛒 P2 · 카페24 (자사몰)

중요
P2

카페24 개발자센터 앱 등록 + OAuth 인증

소요시간 30분 · 난이도 ⭐⭐
필요한 키 5개 — mall_id · client_id · client_secret · access_token · refresh_token
ℹ️ 이 작업은 2단계:
1단계: 앱 등록 → client_id/secret 받기
2단계: OAuth 인증 플로우 한 번 돌려서 access/refresh 토큰 발급

📍 1단계: 앱 등록

1
🔗 카페24 개발자센터 접속 → 쥬얼아이스 운영자 계정으로 로그인
2
"앱 만들기" 클릭 → Public App (공개 앱) 선택
3
앱 기본 정보 입력:
• 앱 이름: OneBoard 매출 연동
• Redirect URI: https://oneboard-server.onrender.com/auth/cafe24/callback
• 앱 종류: 일반 앱
4
권한(Scopes) 체크:
✓ mall.read_application
✓ mall.read_order (주문 조회)
✓ mall.read_product (상품 조회)
✓ mall.read_analytics (통계 조회)
✓ mall.read_store (쇼핑몰 정보)
5
저장 후 앱 상세 페이지에서:
• Client ID 복사
• Client Secret 복사
• mall_id 는 우리 자사몰 URL 앞부분 (예: jewelice.cafe24.com → jewelice)

📍 2단계: OAuth 인증 → 토큰 발급

6
찰스에게 1단계 키 3개 먼저 전달 (mall_id, client_id, client_secret)
→ 찰스가 oneboard-server 설정 후 인증 URL 생성해서 다시 전달
7
찰스가 보낸 인증 URL을 브라우저에 붙여넣고 → "허용" 클릭
8
리다이렉트되면 자동으로 access/refresh 토큰이 서버에 저장됨 (권수지 대리가 직접 복사할 필요 없음)
⚠️ OAuth 인증은 한 번만: 인증을 한 번 통과하면 토큰은 오래(수개월) 유지됩니다. 토큰 만료되면 자동 갱신됩니다.

📘 P3 · META & 네이버 검색광고

광고 ROAS용
P3

META (Facebook/Instagram 광고)

소요시간 20분 · 난이도 ⭐⭐
필요한 키 4개 — app_id · app_secret · access_token · ad_account_id
1
🔗 Meta for Developers 접속 → 쥬얼아이스 계정 로그인
2
"내 앱" → "앱 만들기" → 비즈니스 선택
3
앱 이름 OneBoard 연동, 앱 용도: 광고 관리
4
제품 추가 → Marketing API 활성화
5
설정 → 기본 설정에서 App ID · App Secret 복사
6
Graph API Explorer 접속 → ads_read, ads_management, business_management 권한으로 Long-Lived Access Token 생성
7
비즈니스 관리자 → 광고 계정 → ad_account_id 복사 (act_ 로 시작)
8
4개 키 모두 찰스에게 전달
💡 Tip: 현재 메타 광고 대행사(AMPM글로벌 또는 이전 대행사)가 이미 계정에 접근 중일 수 있습니다. 기존 앱이 있다면 그것을 활용해도 됩니다.
P3

네이버 검색광고

소요시간 15분 · 난이도 ⭐⭐
필요한 키 3개 — api_key · secret_key · customer_id
1
🔗 네이버 검색광고 접속 → 로그인
2
우측 상단 "도구" → "API 사용 관리" 클릭
3
"API 라이선스 발급" 버튼 클릭 → 라이선스 발급
4
발급된 화면에서 복사:
• API Key (긴 문자열)
• Secret Key
• CUSTOMER_ID (광고주 고유 번호)
5
3개 키 찰스에게 전달

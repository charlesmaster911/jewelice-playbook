과장 업무 가이드

쥬얼아이스 · 운영 + 정산 + B2B 실행
과장은 회사 운영 실무를 책임집니다 — 정산·세무·소싱·B2B 운영·공동구매 집행을 직접 처리하고, 팀장에게 정확하게 보고합니다.

책임 영역

1) 정산·세무·인사 운영

  • 월 정산 보고서 작성 (한반도·쥬얼아이스 분리)
  • 4대보험·세무 일정 관리
  • 매입·매출 세금계산서 발행·수령 확인
  • 본인 4대보험 전환 업무 (사업소득 → 4대보험)

2) B2B 운영 지원

  • 기존 파트너 정기 발주 관리 (데블스도어·제이코스·바차·코어인터내셔날·와배송 등)
  • 발주 처리 → 출고 → 수금 사이클 관리
  • 신규 업장 사전 조사 (SNS·리뷰) — 미팅 전 자료 정리
  • 샘플 발송 → 소량 → 정기계약 단계 관리

3) 소싱·MD

  • 원부자재 소싱 (포장재·박스·라벨 등)
  • 거래처 단가 비교·협상
  • 신제품 시제품 단계 협력업체 관리

4) 공동구매·외주 집행

  • 공동구매 진행 시 발주·정산·CS 집행
  • 외주 디자인·영상 발주서 처리
  • 인플루언서 샘플 발송 관리

핵심 KPI

수금 회수익월 말까지 95%+

샘플 발송요청 D+2 이내

매일 / 매주 루틴

매일 09:00 (출근 직후)

  • 어제 발주·출고·수금 사이클 점검
  • 오늘 처리할 정산·발주·소싱 항목 우선순위
  • 팀장(박지현)에게 일일 보고 — 한 줄

매일 18:00 (퇴근 직전)

  • 한 일 · 미완료 · 내일 첫 1개 정리
  • 긴급 미수금·미발송 건 팀장에게 신호

매주 금요일 16:00

  • 주간 정산 마감 점검
  • 다음 주 발주·소싱 일정 확정
  • B2B 미팅 사전 자료 검토

매월 1~5일

  • 월 정산 보고서 작성 → 팀장·대표 보고
  • 세금계산서 발행·수령 확인

사용 시스템

  • 사방넷 — 발주·재고 통합
  • 홈택스 / 더존 — 세무·세금계산서
  • 각 채널 어드민 — 스마트스토어·쿠팡·카카오 정산
  • 은행 인터넷뱅킹 — 수금·송금 확인
  • Gmail / Drive — 거래처 메일·문서
  • Claude Code (이 폴더) — 업무 자동 로그 · 정산표 정리 · 메일 초안

거래처 연락 — 톤·원칙

  • 존댓말 + 정확한 수치 + 명확한 마감일
  • 약속한 일정·금액은 반드시 서면 (메일 또는 카톡 캡처)
  • 가격 협상 → 단독 진행 X. 팀장·대표 승인 후 진행
  • 새 거래처 — 사업자등록증·통장사본 사전 확인

막힐 때

  1. 30분 이상 막히면 — 팀장에게 한 줄 신호 ("블로커: ___ 왜냐하면 ___")
  2. 거래처 가격·계약 조건 → 팀장 또는 대표 승인 필수
  3. 실수·사고 → 숨기지 말고 즉시 보고. 빠른 보고가 빠른 수습
  4. 지급 지연·미수금 30일 초과 → 팀장 즉시 에스컬레이션

매주 금요일 — 점검 5문

  • 이번 주 정산·발주·수금 모두 마감됐나?
  • 지연된 거래처가 있다면 팀장에게 보고됐나?
  • 다음 주 발주·소싱 일정이 명확한가?
  • B2B 미팅 사전 자료가 준비됐나?
  • 다음 주 첫 한 걸음이 명확한가?

```bash
date '+%Y-%m-%d'
```

### 2. 오늘 일정 조회 (Google Calendar MCP)
- `list_events` 호출: startTime=오늘 00:00 KST, endTime=오늘 23:59 KST
- timeZone: Asia/Seoul

### 3. 업무 시트 조회 (Google Drive MCP)
- `read_file_content` fileId: `13yy1MtUhXNg4MEG6qL9lJogIDNdouXqQzv8mkUp2Kkk`
- 찾을 데이터:
  - **박지현** 이름으로 할당된 이번 주 업무 (주간/일간)
  - **권나경** 미완료/놓친 업무
  - **권수지** 미완료/놓친 업무
  - 회의록 최신 의사결정·업무지시

### 4. 현재 HTML 읽기
- `Read` jihyun-daily.html

### 5. HTML 섹션 업데이트 (구조·스타일 유지)

다음 섹션들의 내용만 갱신. CSS/스크립트는 건드리지 않음:

| 섹션 | 업데이트 내용 |
|------|---------------|
| `id="dateChip"` (또는 date-chip 영역) | `YYYY년 M월 D일 (요일) · W주차` |
| 오늘의 一 one-line | 오늘 가장 중요한 한 줄 |
| 오늘 일정 table | 오늘 캘린더 이벤트 시간순 |
| 이번 주 계획 stat-grid + table | W주차 박지현 주간 업무 |
| 이번 달 핵심 목표 milestone-row | 이번 달 남은 핵심 일정 |
| 박지현 오늘 할 일 table | 우선순위순 할 일 |
| 권나경 table | 시트의 권나경 미완료 업무 |
| 권수지 table | 시트의 권수지 미완료 업무 |
| 긴급 주의사항 urgent-box | 이번 주 D-day 긴급 3건 |
| 정부지원 D-day table | 마감 임박순 |
| meta-row 칩 3개 | 오늘의 一 + 다음 질문 + 실행 한 점 |
| `id="genTime"` | `최근 업데이트: YYYY-MM-DD KST` |

## 주의사항

- **이름 기준**: 팀장 직책이 아니라 **"박지현"** 이름으로 시트에서 검색
- **구조 유지**: HTML 구조·CSS·스크립트·편집모드 로직 건드리지 않기
- **Korean**: 모든 콘텐츠는 한국어
- **시트 접근 불가 시**: 그 섹션을 "시트 확인 필요" 노트로 남기고 나머지 진행
- **커밋 실패 시**: 문제 로그 남기고 종료 (repo 상태 유지)

CONCEPT B-1 · MAIN

COURT MAP NAVY

"THE COURT IS THE CANVAS."
3 COLORWAYS · NAVY/WHITE/CHARCOAL

Design

라켓볼 코트 도면을 골드 라인으로 정밀하게 그래픽화. 코트 치수 + GPS 좌표(금천 37.4847°N) 자수. Dieter Rams 무드의 건축적 정밀함.

Marketing Hook

"코트가 곧 옷이 된다" — 라켓볼 클럽 멤버만 알아보는 인사이더 코드. 운동인 정체성과 럭셔리 무드를 동시에.

Specs

네이비 + 골드 / 클린 화이트 + 네이비 / 차콜 + 골드 (3컬러)
실크스크린 + 부분 자수

CONCEPT B-2 · ALTERNATIVE

COURT TOPOGRAPHY

"READ THE GAME."
3 COLORWAYS · BLACK NEON / ICE / CONCRETE

Design

코트 도면을 등고선/히트맵으로 추상화. 랠리의 에너지 흐름을 시각화. 스포츠 데이터 무드의 미래적 라인워크.

Marketing Hook

"경기를 읽는다" — 분석가 같은 포지셔닝. 단순 디자인 넘어 "라켓볼을 사고하는 사람"의 시그널.

Specs

블랙 + 네온그린 (메인) / 아이스 화이트 / 콘크리트 그레이
승화 전사 + 네온 포일

CONCEPT C · HEBE

HEBE GODDESS

"PLAY HARD. STAY ELEGANT."
3 COLORWAYS · BLACK / WHITE / OLIVE

Design

그리스 청춘의 여신 헤베 + 월계관 + 빛나는 태양 심볼. 첨부 레퍼런스 그대로의 모던 럭셔리 라인. 강인함과 우아함의 균형.

Marketing Hook

"승리의 여신을 입는다" — 클럽 정체성 + 자부심. 대회 출전·공식 행사·일상 모두 커버. 가장 안정적이고 고급.

Specs

네이비 + 골드 (메인) / 클래식 블랙 / 클린 화이트 / 올리브 그린
고급 실리콘 나염 + 소매 엠블럼 자수

SNS 콘텐츠 담당 업무 가이드

쥬얼아이스 · 인스타 릴스 · 유튜브 쇼츠 · 콘텐츠 엔진
SNS 콘텐츠 담당은 인스타 릴스 · 유튜브 쇼츠 · 블로그를 통해 쥬얼아이스 브랜드 감성을 전달하고 D2C 매출에 직접 기여합니다 (월 ≥ 20편).

책임 영역

1) 콘텐츠 기획 (월 ≥ 20편)

  • 주간 콘텐츠 캘린더 작성
  • 제품 사용 시나리오 발굴 (홈카페·홈바·선물·B2B 업장)
  • 경쟁 브랜드 모니터링 (월 1회)

2) 제작

  • 인스타 릴스 — 주 5편 이상 (9:16, 30~60초)
  • 유튜브 쇼츠 — 주 2~3편
  • 인스타 피드 — 정방형(1:1) 또는 세로(4:5)
  • 썸네일·카피·해시태그 작성

3) 업로드 + 운영

  • 예약 발행 (피크 시간대: 평일 12시 / 19시 / 21시)
  • 댓글·DM 응대 (24시간 이내)
  • 해시태그 — 브랜드 + 카테고리 + 감성 조합

4) 성과 기록 + 리포트

  • 주간 콘텐츠별 도달·저장·공유·전환 기록
  • 참여율(ER) 3% 이상 유지
  • 월간 TOP 3 / BOTTOM 3 회고

매일 / 매주 루틴

매일 10:00

  • 전날 게시물 성과 확인 (도달·저장·공유·ER)
  • 댓글·DM 1차 응대
  • 오늘 제작·업로드할 콘텐츠 정리

매일 17:00 (제작 마감)

  • 예약 발행 등록
  • 해시태그·카피·썸네일 최종 점검

매주 월요일 11:00

  • 이번 주 콘텐츠 캘린더 확정 (5편 이상)
  • 팀장에게 주간 계획 공유

매주 금요일 16:00

  • 주간 성과 리포트 작성 (TOP 3 / BOTTOM 3)
  • 다음 주 캘린더 초안

매월 25일

  • 월간 ER · 도달 · 팔로워 증가 리포트
  • 잘된 포맷 / 안 된 포맷 분석

제작 체크리스트 (콘텐츠 1편당)

  • 제품이 어떤 상황·감성에서 쓰이는가? (Use case 먼저)
  • 경쟁 제품 대비 차별점이 드러나는가?
  • CTA (구매·팔로우·공유) 명확한가?
  • 해시태그 — 브랜드 + 카테고리 + 감성 조합?
  • 썸네일 첫 1초가 시선 잡는가?
  • 30~60초 안에 메시지 끝나는가? (릴스·쇼츠)
  • 브랜드 컬러(화이트·아이스블루·실버) 유지됐나?

사용 시스템 / 도구

  • 인스타그램 / 유튜브 / 블로그 — 채널 본진
  • Canva / Premiere / CapCut — 디자인·편집
  • Meta Business Suite — 예약 발행 · 인사이트
  • 유튜브 스튜디오 — 쇼츠 분석
  • Claude Code (이 폴더) — 카피 초안 · 해시태그 추천 · 성과 리포트 자동 정리

막힐 때

  1. 30분 이상 아이디어 막힘 → 팀장 또는 Claude Code에 "오늘 릴스 아이디어 5개 줘" 요청
  2. 제품 정보 불확실 → 팀장에게 즉시 확인 (잘못된 정보 게시 금지)
  3. 고객 컴플레인 DM → 단독 응답 X, 팀장 또는 채널톡 CS팀과 공유
  4. 인플루언서 협업 제안 받음 → 팀장 보고, 직접 수락 X
  5. 실수 (오타·잘못된 가격·잘못된 정보) → 즉시 보고하고 게시물 수정·삭제

  • 이번 주 콘텐츠 5편 이상 발행했나?
  • 평균 ER 3% 이상 유지되고 있나?
  • 댓글·DM 24시간 이내 모두 응대했나?
  • 다음 주 캘린더 초안 준비됐나?
  • 이번 주 잘된 콘텐츠 1개 · 안 된 콘텐츠 1개 회고했나?

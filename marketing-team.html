AI 에이전트 팀 (4인)

각 에이전트는 Claude API 기반, 역할별 프롬프트로 분리 동작. 병렬 처리 가능 구간은 동시 실행.

1

🔍 researcher

시장 조사관
홈바·위스키·투명얼음 트렌드 조사, 경쟁사 분석, 타겟 인사이트 도출. 작업 전 context/brand_guide.md · company_info.md 필수 로드.
Output트렌드 리포트 + 키워드 3~5
2

✍️ content_writer

콘텐츠 작가
ablr 톤앤매너 기반 헤드라인·카피·본문 기획. 한국어 기본, 과학 팩트 우선, 감성 + 정보 균형.
Output헤드라인 + 본문 초안
3

🎨 designer

시각 디자이너
HTML 카드뉴스, 뉴스레터, PPTX 슬라이드 구조화. ablr 컬러 시스템 (#1A1A2E · #A8D8EA · #F8FBFF) 엄수.
OutputHTML / PPTX 파일
4

✅ reviewer

품질 검수관
ablr 소문자 표기 검증, 과학 팩트 검수, 오탈자·브랜드 일관성 체크. 마지막 승인 후 output/ 저장.
Output검수 보고서 + 최종 승인

실행 가능 캠페인 메뉴

run_pipeline.py의 --type 옵션. 현재 구동된 캠페인은 없으며, 찰스 지시 시 즉시 실행.

타입 명령어 설명 산출 포맷 처리 에이전트 상태
card_news --type card_news 카드뉴스 (SNS 피드용) HTML R → W → D → Rv 대기
newsletter --type newsletter 월간 뉴스레터 (이메일) HTML R → W → D → Rv 대기
ppt --type ppt 투자·파트너십 프레젠테이션 PPTX R → W → D → Rv 대기
social --type social SNS 포스트 (빠른 단편) MD W → Rv 대기
all --type all 전체 캠페인 (카드뉴스+뉴스레터+PPT) HTML+HTML+PPTX R → W → D → Rv (병렬) 대기
💡 즉시 실행 예시

python run_pipeline.py --topic "투명얼음의 과학" --type all

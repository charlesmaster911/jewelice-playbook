PHASE 1 — 준비
  • 실제 제품 사진 준비 — 아이스볼 64mm, 큐브 50mm, 막대 120×35mm 각각 촬영
  • Character Sheet 1~5 이미지 레퍼런스 Seedance에 업로드
  • 전역 설정 저장 (9:16, 30fps, color grade)
  • Negative Prompt 저장 및 적용 확인
  • Master Style Prompt 준비
PHASE 2 — 훅 검증 (Shot 01~03)
  • Shot 01 생성 — 아이스볼 매크로 플래시 훅 확인
  • Shot 02 생성 — 흐린 얼음 대비 샷 (실제로 불쾌감 주는지 확인)
  • Shot 03 생성 — 아이스볼 드라마틱 등장 확인
  • Shot 01~03 연결 재생 — 훅 효과 검증 후 진행 여부 결정
PHASE 3 — 본 생성 (Shot 04~12)
  • Shot 04 생성 — 아이스볼 아크 샷, Seed 고정
  • Shot 05~07 생성 — 빠른 컷 3연속, 각 0.5초
  • Shot 08 생성 — 피크 샷, 아이스볼 렌즈 플레어 확인
  • Shot 09 생성 — 큐브 50mm, 3면 반사 확인
  • Shot 10 생성 — 막대얼음 3개, 병렬 버블 확인
  • Shot 11 생성 — 3종 플랫레이, 스케일 비교 확인
  • Shot 12 생성 — 브랜드 클로징 크레인 샷
PHASE 4 — 편집 & 완성
  • CapCut Pro / Premiere에서 12샷 타임라인 조립
  • 오디오 타임라인 적용 (Crystal PING → Jazz → Final PING)
  • Text Overlay 추가 (한글 + 영문 병행)
  • 컷 타이밍 정밀 조정 (비트에 맞춰 컷 포인트 이동)
  • Color grade 최종 확인 — 미드나이트 틸 + 앰버 일관성
  • 최종 출력 1080×1920 MP4, 30fps
  • Instagram Reels / YouTube Shorts 업로드

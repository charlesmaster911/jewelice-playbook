4-1. 🏠 내부 100% 생산 시 — 배송 흐름·리스크

순서배송 방법흐름왜 리스크인가
1 퀵 냉동 분산 배송
(드라이아이스 보냉박스)		 사내 냉동고 → 보냉박스 포장 → 퀵서비스 1~5박스 분할 • 퀵 기사 온도관리 불가 · 차량 일반 트렁크

• 각인 얼음 표면 결로 → 각인 흐릿해짐
• 퀵 1건당 3~5만원 × 분할 횟수 → 비용 폭증
2 냉동 택배
(CJ/로젠 냉동)		 사내 포장 → 택배 접수 → 허브 경유 → D+1 배송 • 허브 상차·하차 단계 상온 노출 20~40분

• 택배 허브 파손·분실 사고 시 재생산 불가능(납기 실패)
3 냉동탑차 대절
(정답이지만 비쌈)		 사내 냉동고 → 냉동탑차 -18℃ → 베리쉬 직송 • 수도권 왕복 35~50만원 (1회 기준)
• 1,000개 전량 1회 납품이라 베리쉬 수령 타이밍 일치 필수
• 베리쉬 현장 냉동고 용량 부족 시 수령 거부 가능
4 자차 직접 배송 사내 냉동고 → 팀원 차량 보냉박스 → 베리쉬 전달 • 일반 승용차는 냉장·냉동 기능 없음
• 1,000개 전량 수송 불가 (승용차 보냉박스 한계 50~100개)
• 여름 기온 25℃+ 시 30분 내 녹음 시작
🚨 내부 100% 생산 시 배송 결론: 생산은 가능하더라도 "1,000개를 한 번에 녹이지 않고 옮기는 방법"이 없음.
→ 퀵/택배는 녹음 100% · 냉동탑차는 내부생산 안해도 외주에서 직송 가능하므로 내부 생산의 의미 상실 · 자차는 물리적 캐파 부족.
→ 결국 냉동탑차 쓸 거면 외주 공장이 쓰는 게 더 싸고 안전함.

■ 베리쉬 각인얼음 1,000개 납품 건 — 검토 결과 보고

대표님, 베리쉬 각인얼음 1,000개 납품 건 최적안 검토 완료했습니다.

[결론]
하이브리드 방식(외주 800개 + 사내 200개) + 금형비 별도청구를 제안드립니다.
→ 베리쉬 제시가 개당 6,500원 × 1,000개 = 650만원 + 금형비 별도 80만원.
→ 실질 원가 약 329만원 (기존 인력·냉동고 3대 활용) → 순이익 321만원 + 금형비 마진 20만원 = 약 341만원.

[왜 하이브리드인가]
① 내부 100% 생산은 불가 — 냉동고 3대 풀가동해도 40일간 최대 600개. 1,000개 물리적 한계.
② 외주 100%는 QC 리스크 — 부산 공장에서 베리쉬로 직송하면 사내 검수 기회 없음. 각인 불량 시 대응 불가.
③ 하이브리드는 양쪽 장점만 취함 — 외주로 대량(800개) 빠르게 찍고, 사내 냉동고 3대에 입고해서 QC·재포장·불량 보충(200개)까지 해결. 납기 버퍼 1~2주 확보.

[배송이 핵심]
내부 100%로 만들어도 "1,000개를 녹이지 않고 옮기는 방법"이 없음 — 퀵·택배는 녹음, 냉동탑차 대절하면 외주 공장이 쓰는 게 더 쌉니다. 그래서 내부 생산은 배송 단계에서 반드시 막힙니다.
→ 하이브리드: 장거리(부산→사내)는 공장 냉동탑차, 단거리(사내→베리쉬)는 QC 끝낸 완성품만 깨끗하게 배송.

[다음 액션 — 오늘 중 처리]
① 베리쉬에 확인: 정확 납기일 / 납품주소 / 현장 냉동고 용량 / 로고 벡터파일 / 저작권 동의서
② 금형비 80만원 별도 청구 방침 안내 (재발주 시 50% 환급 조건 카드 제시)
③ 외주 3곳(부산 빅볼 / 아이스팜 / VONESTIS) 견적 요청

[마지노선]
• 최초 제시: 8,000원 (마진 59%)
• 정상 제시: 6,500원 (마진 49%) ← 권장
• 최저 수용: 4,500원 (마진 27%)
• 4,500원 이하 절대 거절 (기회비용 손실)

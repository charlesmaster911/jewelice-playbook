🧠 7단계 · 메모리 초기 세팅 (취향·시그니처) STEP 7

화경은 대화 한 번 한 번을 기억합니다. 첫날 아래 5가지를 등록해두면 매일 같은 말 반복 안 해도 됨.

등록할 것화경에게 이렇게 말하기
호칭"이 방에서 너는 화경이야. 나는 박지현 팀장이고. 찰스 호칭 절대 금지."
시그니처 컬러"내 시그니처 컬러는 퍼플(#7c3aed) + 옐로우(#fbbf24)야. 모든 디자인에 이거 써."
포맷 취향"서술형 텍스트 싫어해. 정리는 항상 표로 만들어줘."
HTML 영구 URL"HTML 만들면 무조건 GitHub Pages 영구 URL로 배포해. 같은 주소에 덮어쓰기."
저장 위치"내 작업물은 무조건 박지현-팀장 폴더에 저장. 다른 데 저장 금지."
💡 박지현 팀장 방은 이미 위 5가지가 모두 메모리 등록되어 있습니다. 새 팀원만 처음에 한 번 입력하면 됩니다.

🔌 8단계 · MCP 연결 (Gmail · Calendar · Drive · Notion) STEP 8

MCP = 화경이 외부 서비스(Gmail·캘린더 등)에 직접 접근하게 해주는 다리. 이거 연결하면 "오늘 일정 알려줘" "안 읽은 메일 정리해줘"가 한 번에 됨.

서비스연결 방법박지현 팀장 활용
Gmail claude.ai → Settings → Integrations → Gmail "Connect" 모닝 브리핑 메일 자동 작성, 답장 초안
Google Calendar Settings → Integrations → Calendar "Connect" 일정 자동 등록, 데일리 리포트 이벤트
Google Drive Settings → Integrations → Drive "Connect" 스프레드시트 직접 읽고 분석
Notion Settings → Integrations → Notion "Connect" (선택) 회의록·문서 연동
Canva Settings → Integrations → Canva "Connect" (선택) SNS 콘텐츠·썸네일 제작
💡 한 번 연결하면 다음 로그인부터 자동. 회사 계정으로 연결하는 것이 좋습니다 (개인 메일 섞이지 않게).
